### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mahmoud-talbawy/My_Portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd My_Portfolio
   ```

3. **Checkout the FlutterFlow branch** (if applicable)
   ```bash
   git checkout flutterflow
   ```

4. **Install dependencies**
   ```bash
   flutter pub get
   ```

5. **Run the app locally**
   ```bash
   flutter run -d chrome
   ```
   *(Use `-d chrome` for web preview)*

6. **Build for production**
   ```bash
   flutter build web
   ```

## ğŸŒ Deployment

### Deploy to GitHub Pages

1. Build the web version:
   ```bash
   flutter build web
   ```

2. Deploy the `build/web` folder to GitHub Pages

### Deploy to Firebase Hosting

1. Install Firebase CLI:
   ```bash
   npm install -g firebase-tools
   ```

2. Initialize and deploy:
   ```bash
   firebase init hosting
   firebase deploy
   ```

### Deploy to Netlify

1. Connect your GitHub repository to Netlify
2. Set build command: `flutter build web`
3. Set publish directory: `build/web`
4. Deploy automatically on push
