Mahmoud Portfolio
A responsive personal portfolio website built with Flutter and FlutterFlow to showcase skills, projects, and contact information in a clean, modern UI.​

Features
Single-page responsive layout optimized for web, tablet, and mobile.​

Sections for About, Skills, Projects, and Contact with smooth navigation.​

Built visually in FlutterFlow with generated Flutter code for full customization.​

Tech stack
Flutter & Dart for the application code.​

FlutterFlow as the visual builder and project generator.​

GitHub for version control and collaboration.​

Getting started
Clone the repository: git clone https://github.com/mahmoud-talbawy/My_Portfolio.git and checkout the flutterflow branch.​

Open the project in FlutterFlow (import/export as needed) or run the Flutter code locally with flutter run.​

Update the content (name, bio, skills, project links) directly in FlutterFlow or in the Flutter widgets.​

Project structure
flutterflow/ – auto-generated Flutter project from FlutterFlow.​

lib/ – main Dart code (widgets, pages, routing, theming).​

assets/ – images, icons, and other static resources used in the portfolio UI.​

Customization
Change colors, typography, and spacing via the FlutterFlow theme or the Flutter ThemeData configuration.​

Add or remove sections (e.g., Blog, Services, Testimonials) by creating new pages/sections in FlutterFlow and wiring navigation.​

Integrate backends (Firebase, REST APIs, etc.) for dynamic content such as blogs or contact forms.
